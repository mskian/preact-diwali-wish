## Setup and Usage

- Clone or Download the Repo

```sh
git clone https://github.com/mskian/preact-diwali-wish
cd preact-diwali-wish
```

- Install dependencies

```sh
pnpm install
```

- Test

```sh
pnpm dev
```

- Build the App

```sh
pnpm build
```
